# ClaimAudit AI

Healthcare Claims Reimbursement Audit Agent.

## Features

- Upload claims CSV
- Validate claims
- Detect anomalies
- Risk scoring
- Dashboard analytics
- Report export

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app/main.py
```

## Stack

- Streamlit
- Pandas
- Scikit-Learn
- Plotly
- SQLite
